Manual Setup

Alternative: Manual Installation

  1. Right-click your bookmarks bar and select Add page
  2. Set the name to: Axiom Alerts
  3. Click the URL field and paste the copied script (Ctrl+V)
  4. Click Save to create the bookmark
  5. Navigate to axiom.trade and log into your account
  6. While on Axiom, click the Axiom Alerts bookmark
  7. Your alert feed is now active — real-time data will begin streaming
